Elevator Game

A fun, personal project.

The concept is simple: you’re "stuck" on the top (ninth) floor and need to work your way down to the ground floor. But there's a twist — to go down each floor, you must complete the task. Each floor prompt is a task or goal you've been meaning to accomplish. You fill in the prompts with tasks you've been avoiding before playing the game, and then "play" the game by going floor by floor, completing each task.

Technologies Used:

Frontend: JavaScript, HTML, CSS
